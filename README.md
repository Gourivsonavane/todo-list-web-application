# To-Do List Web Application

A web-based To-Do List application that allows users to create, manage, and track their daily tasks through a simple and responsive interface.

## 🚀 Features

- Create, Read, Update, and Delete (CRUD) operations for tasks
- Responsive and clean UI built with HTML, CSS, and Bootstrap
- Form validation to ensure valid task entries
- Persistent storage of tasks using SQL Server

## 🛠️ Technologies Used

- **Frontend:** HTML, CSS, Bootstrap
- **Backend Framework:** ASP.NET Core MVC
- **Database:** SQL Server
- **ORM:** Entity Framework Core
- **Architecture:** Model-View-Controller (MVC)

## 📋 What This Project Demonstrates

- Building a full-stack web application using ASP.NET Core MVC
- Designing a responsive front-end with Bootstrap
- Connecting to SQL Server using Entity Framework Core
- Implementing basic form validation for data integrity
- Testing application functionality and resolving bugs during development

## ⚙️ How to Run Locally

1. Clone the repository
   ```bash
   git clone https://github.com/https://github.com/Gourivsonavane/todo-list-web-application.git
   ```
2. Open the solution file (`.sln`) in Visual Studio
3. Update the connection string in `appsettings.json` to point to your local SQL Server instance
4. Run Entity Framework Core migrations (if applicable):
   ```bash
   dotnet ef database update
   ```
5. Build and run the project (F5 in Visual Studio, or `dotnet run` from CLI)
